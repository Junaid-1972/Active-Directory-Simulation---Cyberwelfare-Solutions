# Active-Directory-Simulation---Cyberwelfare-Solutions
Identity and Access Management (IAM) Project Active Directory (On-Premises) Deployment For CyberWelfare Solutions.
Installing and hardening IAM / Active Directory align with NIST Cyber Security Framework (CSF) IDENTIFY function ID.AM (Asset Management) Identifying and managing accounts, roles, and privileges tied to organizational assets.
PROTECT function specifically PR.AC(Access Control) ensuring only authorized users, devices, and processes can access assets. and PR.PT(Protective Technology). In ISO 27001 it maps closely to Annex A.5 (Organizational Controls) + Annex A.8 (Technological Controls) which is about access governance and technical enforcement
